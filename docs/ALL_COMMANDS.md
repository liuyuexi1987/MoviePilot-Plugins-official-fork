# Agent影视助手命令总览

这份文档只保留当前建议继续使用的主线命令。

如果你是新手，先记住一个原则：

- 先搜
- 再回编号
- 不要依赖太多别名和高级命令

---

## 最常用

| 命令 | 作用 |
|---|---|
| `搜索 <片名>` | 默认优先走 MP/PT；如果 MP/PT 已关闭，再按当前启用源回退 |
| `MP搜索 <片名>` | 走 MoviePilot 原生 PT 搜索 |
| `PT搜索 <片名>` | 同 `MP搜索` |
| `盘搜搜索 <片名>` | 先查盘搜；盘搜没结果时按开关补查影巢 |
| `影巢搜索 <片名>` | 先查影巢；影巢没结果时按开关补查盘搜 |
| `下载 <片名>` | 先识别媒体，再走 MoviePilot 原生 PT 下载链 |
| `更新检查 <片名>` | 检查当前媒体是否有更新资源 |
| `检查 <片名>` | 同 `更新检查` |

补充说明：

- `搜索 第4集`、`搜索 最新集` 这类带集数线索的写法，会直接按 MP/PT 搜索。
- `下载 <片名>` 如果片名有歧义，会先让你选正确影片或剧集。
- 标题级 `转存 <片名>` / `115转存 <片名>` / `夸克转存 <片名>` 已取消。现在统一建议先搜索，再从结果里回编号继续处理。

## 选编号

| 命令 | 作用 |
|---|---|
| `1` | PT 结果里直接下载；云盘结果里继续处理第 1 条 |
| `选择 1` | 同 `1` |
| `选择 1 详情` | 查看第 1 条详情 |
| `详情 1` | 同 `选择 1 详情` |
| `n` | 下一页 |
| `下一页` | 同 `n` |

补充说明：

- PT 结果通常直接回编号下载，不提供聊天里的详情卡。
- 云盘/影巢结果如果要复制完整信息或链接，再用 `选择 1 详情`。

## 分享链接

直接发送分享链接即可，无需额外前缀：

| 输入 | 效果 |
|---|---|
| `https://115.com/s/xxxxx` | 自动走 115 处理链 |
| `https://pan.quark.cn/s/xxxxx` | 自动走夸克处理链 |

如有提取码，也可以一起发。

## 已取消的入口

下面这组命令不再建议继续用：

- `云盘搜索 <片名>`
- `转存 <片名>`
- `115转存 <片名>`
- `夸克转存 <片名>`
- `推荐`
- `热门推荐`
- `智能发现`
- `热门发现`
- `资源决策`
- `智能决策`

现在更推荐的做法是：

1. 明确写来源去搜
   例如：`盘搜搜索 片名` / `影巢搜索 片名` / `PT搜索 片名`
2. 看结果
3. 直接回编号继续
